A comparison Between Blaze.js and React
=======

## What and Why

* Compare Blaze v React based on the following criteria
* Performance
* Scalability
* Maintainability
* Browser Support
* Mobile Device Performace(Render speed and Animation)
* Ease of Use

## Running it

* bower install

* npm start
